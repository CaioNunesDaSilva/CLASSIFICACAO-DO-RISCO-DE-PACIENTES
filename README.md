# CLASSIFICACAO DO RISCO DE PACIENTES
Projeto de TCC para graduação do curso de ciência da computação na Universidade Paulista.
O principal objetivo dessa pesquisa é o desenvolvimento de um equipamento protótipo para o uso hospitalar, consistindo em um dispositivo Arduino com um sensor para medir o nível de oxigenação do sangue, frequência cardíaca e temperatura de um paciente antes da triagem, enviara os dados a uma aplicação executada em um servidor, a qual irá categorizar o nível de risco do paciente com base nos sinais vitais recebidos.
